# React Background Changer

A simple background color changer built using React and Tailwind CSS.

## Features

- 🎨 Change background color with one click
- ⚛️ Built with React
- 💨 Styled using Tailwind CSS
- ⚡ Powered by Vite

## Tech Stack

- React
- Tailwind CSS
- Vite

## Installation

```bash
npm install
npm run dev
```
